## Práctica con javaEE usando el patrón MVC

La idea es hacer una ABM de personajes de dragon ball donde cada personaje contará con su imágen, nombre, descripción y técnica especial.

### Funcionalidades:
- Visualizar personajes
- Agregar personaje
- Eliminar personaje
- Modificar personaje

### Herramientas:
- Eclipse Java Neon
- Tomcat 9.0
- Mysql
- Bootstrap
